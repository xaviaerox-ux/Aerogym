# 🏋️ AeroGym: Tu Entrenador Antigravedad Personal

AeroGym es una aplicación web progresiva (PWA) diseñada para optimizar tu rendimiento físico mediante la integración de datos de salud, rutinas inteligentes y análisis de IA.

## ✨ Características Principales

- **🔄 Rotación Inteligente PPL**: Sistema automático que gestiona tus sesiones de Empuje (Push), Tracción (Pull) y Pierna (Legs) para asegurar un progreso equilibrado.
- **📊 Universal Health Ingestion**: Importa tu historial de **Xiaomi (Zepp/Mi Fitness)** y **Google Fit** de forma acumulativa para visualizar tu evolución a largo plazo.
- **🛡️ Privacidad Total**: Todos tus datos se procesan y almacenan localmente en tu navegador. AeroGym no utiliza servidores externos para tus datos personales.
- **🧠 Readiness Engine**: Un algoritmo que analiza tu sueño y actividad para recomendarte la intensidad ideal de tu próximo entrenamiento.
- **🤖 Aero AI Coach**: Consejos personalizados y análisis de progresión utilizando inteligencia artificial.

## 🚀 Instalación y Uso

**Requisitos:** Node.js (v18+)

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/Aerogym.git
   cd Aerogym
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. (Opcional) Configura tu `GEMINI_API_KEY` en `.env.local` para habilitar funciones de IA.
4. Arranca la aplicación:
   ```bash
   npm run dev
   ```

## 🛠️ Tecnologías

- **Vite + React + TypeScript**
- **Tailwind CSS** para un diseño premium y responsivo.
- **Lucide Icons** & **Framer Motion** para una UX fluida.
- **Recharts** para visualización de datos de alto rendimiento.
